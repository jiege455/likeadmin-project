> QQ: 2711793818 杰哥

## 一、问题背景

前端页面导航存在以下问题：
1. **返回功能不稳定**：部分页面返回按钮无法正常返回上一页
2. **标题不居中**：很多页面标题没有正确居中显示
3. **样式不统一**：各页面导航样式不一致，有的使用白色背景，有的使用主题色背景
4. **代码重复**：每个页面都需要写大量重复的导航代码

## 二、解决方案

### 2.1 创建统一导航组件

创建了 `custom-nav` 组件，位于：
```
uniapp/src/components/custom-nav/custom-nav.vue
```

### 2.2 组件特性

1. **自动适配状态栏高度**：自动获取系统状态栏高度，适配不同设备
2. **标题自动居中**：使用绝对定位确保标题始终居中
3. **智能返回逻辑**：
   - 优先返回上一页
   - 如果没有上一页，自动跳转到首页或指定页面
   - 支持 tabBar 页面自动使用 switchTab
4. **主题适配**：自动读取主题配置，支持自定义背景色和文字颜色
5. **右侧插槽**：支持右侧自定义内容（如添加按钮、编辑按钮等）

### 2.3 使用方法

```vue
<!-- 基础用法 -->
<custom-nav title="页面标题"></custom-nav>

<!-- 自定义颜色 -->
<custom-nav title="页面标题" text-color="#ffffff"></custom-nav>

<!-- 带右侧按钮 -->
<custom-nav title="页面标题">
    <template #right>
        <view class="w-10 h-full flex items-center justify-center" @click="handleAdd">
            <u-icon name="plus" size="20"></u-icon>
        </view>
    </template>
</custom-nav>

<!-- 隐藏返回按钮 -->
<custom-nav title="页面标题" :show-back="false"></custom-nav>

<!-- 自定义返回页面 -->
<custom-nav title="页面标题" default-url="/pages/index/index"></custom-nav>
```

## 四、组件注册

在 `pages.json` 中已注册组件：
```json
{
    "easycom": {
        "custom": {
            "^custom-nav$": "@/components/custom-nav/custom-nav.vue"
        }
    }
}
```

## 五、开发规范

### 5.1 新增页面规范

1. 所有需要导航栏的页面，必须使用 `custom-nav` 组件
2. 不再使用 `page-meta` + `navigation-bar` 的组合
3. 不再手动编写 H5 端的固定导航栏代码

### 5.2 命名规范

- 组件名称：`custom-nav`（kebab-case）
- 使用时：`<custom-nav></custom-nav>`

### 5.3 样式规范

1. 默认使用主题配置的背景色和文字颜色
2. 需要白色文字时，添加 `text-color="#ffffff"` 属性
3. 页面内容区域不需要再添加导航栏占位高度

## 六、注意事项

1. **返回逻辑**：组件已内置智能返回逻辑，无需手动编写 `goBack` 方法
2. **状态栏适配**：组件自动处理状态栏高度，无需手动计算
3. **主题切换**：组件自动响应主题变化，无需手动处理
4. **H5兼容**：组件同时支持 H5 和小程序端，无需条件编译
