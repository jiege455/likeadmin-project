## 📁 文件列表

在 `server/public/install/db/` 目录下有以下文件：

### 1. **like.sql** - 完整安装包
- **大小：** ~137 KB
- **内容：** 69 个表结构 + 完整初始化数据
- **用途：** 一键安装系统（包含所有配置和菜单数据）
- **适用场景：** 
  - 新服务器部署
  - 完整系统安装
  - 需要预设数据的场景

### 2. **like_structure.sql** ⭐ - 纯表结构文件
- **大小：** ~70 KB
- **内容：** 仅 69 个表结构（不含任何数据）
- **用途：** 创建空数据库结构
- **适用场景：**
  - 需要自定义数据
  - 开发环境搭建
  - 数据迁移（只迁移结构）
  - 版本升级（只更新表结构）

### 3. **db_patch.sql** - 补丁文件
- **内容：** 补充 2 个重要表（la_user_series_permission, la_open_result_log）
- **用途：** 如使用旧版 like.sql，需要补充这两个表

## 🎯 使用说明

### 方案一：使用 like.sql（完整安装）

**适用：** 新服务器一键安装

```sql
-- 在宝塔面板 phpMyAdmin 中执行
-- 导入 like.sql

-- 步骤：
-- 1. 创建数据库
-- 2. 打开 phpMyAdmin
-- 3. 选择数据库
-- 4. 点击"导入"
-- 5. 选择 like.sql 文件
-- 6. 点击"执行"
```

**导入后：**
- ✅ 69 个表结构

- ✅ 系统配置数据
- ✅ 字典数据
- ✅ 部门数据
- ✅ 装修页面数据

## ⚠️ 重要提示

### 使用 like_structure.sql 注意事项

1. **必须手动创建管理员账号**
   ```sql
   -- 示例：创建默认管理员
   INSERT INTO `la_admin` (`id`, `root`, `name`, `avatar`, `account`, `password`, `login_time`, `login_ip`, `multipoint_login`, `disable`, `create_time`, `update_time`) VALUES
   (1, 1, '超级管理员', '', 'admin', MD5('admin123'), NULL, '', 1, 0, UNIX_TIMESTAMP(), UNIX_TIMESTAMP());
   ```

2. **必须导入菜单数据**
   - 否则后台登录后看不到任何菜单
   - 菜单数据 213 条
   - 包含所有功能权限

3. **必须导入系统配置**
   - 否则系统无法正常运行
   - 包括支付配置、网站配置等

### 推荐使用 like.sql

**强烈建议使用 like.sql 进行一键安装！**

原因：
- ✅ 包含所有必要数据
- ✅ 菜单完整（213 条）
- ✅ 配置完整
- ✅ 开箱即用
- ✅ 避免手动配置错误
