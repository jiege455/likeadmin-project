# 文章详情提示配置功能规划

## 1. 功能概述

### 1.1 目标
在文章详情页面增加两个可配置的提示区域：
- **顶部提示区**：位于文章标题下方，用于显示"注意事项"等提示信息
- **底部提示区**：位于文章内容底部，用于显示"购买须知/购买指南"等信息

### 1.2 业务场景
- 平台管理员可以在后台统一配置文章详情页的提示内容
- 支持富文本编辑，可添加图文并茂的说明
- 配置实时生效，无需修改代码

### 1.3 与现有系统关系
- 复用现有的配置管理系统（ConfigService）
- 在后台"网站设置"模块中新增配置项
- 前端文章详情组件读取配置并显示

## 3. 数据库设计

### 3.1 配置存储方式
使用现有的 `la_config` 表存储配置，无需新建表。

### 3.2 配置项
| 配置类型 | 配置名称 | 配置键名 | 说明 |
|---------|---------|---------|------|
| article_tips | top_tips | 顶部注意事项 | 富文本内容 |
| article_tips | top_tips_show | 顶部提示显示开关 | 1显示/0隐藏 |
| article_tips | bottom_tips | 底部购买须知 | 富文本内容 |
| article_tips | bottom_tips_show | 底部提示显示开关 | 1显示/0隐藏 |

- 配置保存成功
- 前端正确显示配置内容
