📧 邮箱配置菜单导入工具

开发者：杰哥网络科技 QQ: 2711793818

📌 工具说明：
本工具用于在后台管理系统中添加"邮箱配置"菜单，让您可以在后台配置邮箱服务。

🎯 快速执行（推荐）

第一步：执行导入

点击下方按钮，自动执行 SQL 导入，将邮箱配置菜单添加到数据库。

第二步：清除缓存

导入成功后，需要清除系统缓存才能看到新菜单。

🔧 前往清除缓存

第三步：验证菜单

点击下方按钮验证菜单是否成功添加。

✓ 验证菜单

📋 菜单详情

字段 值
菜单名称 邮箱配置
菜单位置 系统设置 → 邮箱配置
菜单图标 el-icon-Message
权限标识 setting.email/config
组件路径 setting/email/index

⚙️ 邮箱配置说明

⚠️ QQ 邮箱注意事项：
QQ 邮箱需使用授权码，而不是登录密码。授权码获取方法：
1. 登录 QQ 邮箱网页版 → 设置 → 账户
2. 开启"POP3/SMTP 服务"
3. 生成授权码
配置项 说明 示例
SMTP 服务器 邮箱 SMTP 服务器地址 smtp.qq.com
端口 SMTP 服务器端口 465 (SSL)
加密方式 SSL / TLS / 无加密 SSL
发件人账号 发送邮件的邮箱账号 your@qq.com
授权密码 邮箱授权码（非登录密码） xxxxxxxxxxxx
管理员邮箱 接收系统通知的邮箱 admin@example.com

✅ 验证清单

  • 执行导入脚本
  • 查看执行结果（显示成功信息）
  • 清除系统缓存
  • 重新登录后台
  • 在"系统设置"菜单下找到"邮箱配置"
  • 配置邮箱服务信息
  • 发送测试邮件验证
